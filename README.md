Movie Dashboard (Django)

Quick start:
1. Create virtualenv and install requirements: pip install -r requirements.txt
2. Copy .env.example to .env and set API keys
3. Run migrations: python manage.py migrate
4. Run server: python manage.py runserver
5. (Optional) Crawl H5AI: python manage.py crawl_h5ai --url "https://your-h5ai/files/"
